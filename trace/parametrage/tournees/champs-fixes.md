# Champs fixes

Les champs fixes permettent de choisir quels champs sont disponibles sur le formulaire de création et de modification d'une tournée. Ils sont paramétrables par Type : cliquez sur le Type souhaité pour paramétrer les champs.

Pour chaque champ, choisissez les éléments suivants :&#x20;

* **Formulaire de création :**&#x20;
  * **Afficher** : cochez la case pour que le champ soit présent sur le formulaire de création.
    * **Obligatoire** : cochez la case pour qu'il ne soit pas possible de créer une tournée sans que ce champ soit renseigné sur le formulaire de création.
* **Formulaire d'édition :**&#x20;
  * **Afficher** : cochez la case pour que le champ soit présent sur le formulaire de modification.
  * **Obligatoire** : cochez la case pour qu'il ne soit pas possible de valider la modification d'une tournée sans que ce champ soit renseigné sur le formulaire de modification.
